Administration

Tesseract

n order to install Tesseract (which delivers the core OCR capabilities) on Ubuntu or Debian you can just install it via apt-get:

sudo apt-get install tesseract-ocr

After the installation of tesseract you only will have the english language available. If you want to have more options, you can install an additional language package like follows:

sudo apt-get install tesseract-ocr-deu

For further information look on:

  • Compilation
  • Pre-built package

Please consider that the ocrmypdf as well as the tesseract command have to be available for the webserver user.
