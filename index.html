OCR

Nextcloud OCR processing for images and PDF with tesseract-ocr, OCRmyPDF and Gearman for asynchronous purpose.

View on GitHub Download .zip Download .tar.gz

Installation

Install the app from the Nextcloud Appstore or download the git release and place the content in owncloud/apps/ or nextcloud/apps/.

Basic usage

Just click on the menu for a specific file (image or pdf), which you want to be processed or select multiple files (images and/or pdfs) and click on OCR. A little submenu shows up and you have to select a language for the ocr processing engine, then click "Process" and wait until the processing finishes. You will be informed about the status on the top of the screen (notification bar). The Process will create new files called "_OCR.txt" or "_OCR.pdf" appended to the original name.

Screenshots and example usage

The OCR app gives you much more possibilities for your personal organization. With OCR you will be able to process all your images and pdf files in order to get the characters in it optically recognized and transformed to a txt file or another pdf which has now a text layer, allowing it to be searched. The great benefit of OCR: It is processing absolutely asynchronous in the background.
